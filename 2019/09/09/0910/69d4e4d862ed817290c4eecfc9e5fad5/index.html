『 葡萄牙 』阿尔加维海岸线自驾之旅

发表于 2019-09-09 |

葡萄牙 不止有 里斯本 和 辛特拉
，这里从南到北有许多精美的小镇，壮阔的自然景观，大自然的鬼斧神工在这片土地上演绎得淋漓尽致，在阿尔加维这片海岸线上造就了许多令人惊叹的悬崖洞穴 和美 景。
埃武拉 、奥比都斯等古老的历史名城同样吸引着无数游人来此一探究竟，感受历史遗留下的痕迹

埃武拉的历史故事

即便坐惯了洲际航班，抵达 里斯本 依然感觉昏昏沉沉，租好车后便在机场边上的酒店休息了一晚。一夜的安睡，在阳光明媚的早晨，终于开始了此次 葡萄牙 的探秘之旅。

此行的第一站选在位于 里斯本 东南约130公里处的小镇： 埃武拉 ，因正好位于一路向南的途中，在此停留了几个小时以步行的方式参观小镇的一些重要景点。

埃武拉 小镇拥有悠久的历史，追溯到 罗马 帝国时代，始建于公元3世纪的 罗马 帝国统治时期，迄今仍保存着四千多栋 罗马 帝国时代以来不同时期的古建筑， 罗马
风格、哥特式、巴 洛克 风格、文艺复兴时期的历史遗址，被称为“博物馆城市”，早在1986年就被选入《世界文化遗产》名录。

历史上的 埃武拉 是宗教和文化中心，不同时期的建筑给她繁华的过去烙下了精彩的历史印记，1755年 葡萄牙 史上最重大的大地震也未曾毁灭 埃武拉
的过去，甚至以“自1755年大地震后仍完好保存着的 葡萄牙 黄金年代城市范例”这一原因， 成功 入选世界文化遗产。因此，如果想要了解全部的 葡萄牙
历史与文化，不妨来 埃武拉 看看吧。

小镇中的街道纵横交错，大大小小的巷子中分布着各种美食店铺，作为旅游城市，同样还有许多卖小工艺品的纪念品店。

== 人骨教堂 ==

全名圣弗朗西斯科教堂，是 埃武拉 从默默无名到声名远扬的重要因素，建于1510年，是一座集哥特式、曼努埃尔式和巴 洛克
式各种风格的教堂。然而这座教堂真正著名的是其西厅的小人骨教堂，其内从墙壁到柱子均由人骨堆叠而成。和 捷克
的人骨教堂一样，14世纪的一场黑死病夺走了小镇上多数人的生命，太多的无处安放的骨骸最终就成了小教堂中的装饰品，同时也是一种对于生命的敬畏。

由于是周末，人骨教堂仅早上开放，午后来此已然关门了，没有机会进去参观，只能在外面看看教堂的外观，同时教堂前方有一片小广场，是自驾游游客停车的主要区域。

== 埃武拉 大教堂 ==

这座始建于12世纪的教堂位于 埃武拉 最中心地带，也是 埃武拉 镇上最高的建筑，周身由灰色花岗岩砌成，是一座拥有绝妙回廊和堆满教会珍宝的博物馆，也是 葡萄牙
境内最大的中世纪教堂。

大教堂从远处看显得十分敦实厚重，侧面看过去更像是一座堡垒，其正门装点着14世纪雕刻的12圣徒造型，是 葡萄牙 哥特式雕塑大师 梅斯 特佩罗的杰作。

参观大教堂一定要先从顶层的屋顶 天台 开始，通过一段狭小的塔楼楼梯回旋向上，可以看到这座古老教堂的屋顶，彷如城墙般的壮观顶层，还可以一览整个 埃武拉
老城的全景。

埃武拉 老城的古朴建筑，典型的红瓦白墙，于此可以欣赏 埃武拉 360度全景

教堂另一处精美的去处，幽静的庭院 和美 妙的回廊，配上古朴的天主教建筑风格，不经意间纷纷放慢了脚步。

== 古 罗马 狄安娜神庙 ==

埃武拉 另一处著名的景点便是位于大教堂一旁的古 罗马
神庙，神庙前方是一片开满鲜花的广场：狄安娜花园，许多游客和当地人来此静坐、看书、晒太阳。花园后是一座观景平台，适合在此凭栏远眺，鸟瞰古城。

神庙建于公园一世纪，是一座供奉古 罗马 月亮女神和狩猎女神狄安娜的地方，是 伊比利亚半岛 迄今保存最好的 罗马 神庙，更是 葡萄牙 独一无二的古 罗马
神庙。虽然神庙仅存的遗址并不算很壮阔，一米多高的底座，如今仅残留着了14根花岗岩 科林斯 式 石柱 和部分横梁， 石柱
每根高达7.68米，其余的部分早已消失在历史的长河当中。

阿尔加维海岸线奇幻之旅

远离 葡萄牙 城区的南部海岸线阿尔加维，一直以来就仿佛是一片未经开发的秘境，这里有大自然的鬼斧神工打造的神奇海岸线，而15世纪的 葡萄牙
人更是从这里出发踏上了史诗般的征程。

阿尔加维是 葡萄牙
南部的一个大区，以其独特构造的黄金石灰岩悬崖海岸而著称，长达数十公里的石灰岩海岸行程的悬崖礁石为我们展现了无比奇妙的海洋地质奇观，拱门、海蚀洞穴和石灰岩柱随处可见，无论是步行其上，亦或是乘船远观，甚至自己划上一艘皮筏艇穿梭于这些奇特的岩石之间都可以清楚地观赏大自然的奇迹。

我们的酒店位于阿尔加维地区最著名的两个小城拉哥斯与 法鲁 之间，CI后趁着天色未晚，立刻驱车穿越一条条狭小的乡间小道，来到此行最重要的海滩
Albandeira Beach，也是阿尔加维著名的象鼻悬崖，更是摄影师的最佳去处。

在阿尔加维到处是高品质的海滩，有的沙滩一望无垠，有的则在金色峭壁包围之下，甚至还有许多荒芜的小岛和隐秘的海湾。从空中俯瞰海边的悬崖，还可以看到呈现出深浅不一蓝色的北大西洋。

夕阳亦或是 朝阳 中的悬崖，仿佛被染上一层金色的外衣，高耸的巨石和奇形怪状的悬崖峭壁，使得这里就像是一片外星世界。

波尔蒂芒海盗船之旅

在阿尔加维的几天，不能光顾着拍照、看海滩，还提前在网站上预定了半天的出海游玩，即乘船近距离观赏海岸线的礁石。这里每天有许许多多的出海游船行程，不同的小镇，不同的出发地点，根据我们所住的地区选择了Portimao小镇出发，有趣的是居然还有一家将古老的木质海盗船改造过后的出海行程，从小有一个海盗梦的我毫不犹豫的选择了这个行程。

波尔 蒂芒是阿尔加维地区的第二 大城 市，从历史上看， 波尔 蒂芒与阿尔加维一度繁荣的罐头厂息息相关，现如今的 波尔 蒂芒却已 成功
地转型成一个阿尔加维海岸线上著名的旅游目的地。城市里的建筑多以清新亮丽的色彩呈现在世人眼前，白色的房屋一望无际，海岸边则矗立着一栋栋红色、黄色的亮色建筑物，包括博物馆、剧院等，加上紧密排布着的游轮港口，使得这座城市充满了活力和文化底蕴。

没有预留太多的时间参观小镇，驱车停靠在港口边上的免费停车场，远远就看到与周边游轮、快艇显得有点格格不入的木质海盗船，船头的海盗旗迎风飘扬，这就是我们此行半日海上游玩的船了。

要前往该洞穴有三种方式：

1. 根据潮汐，如果大海平静，可以直接从海滩游泳到洞穴 ，但有一定危险，非水中小白龙勿随意尝试；

2.
每天都有的乘船游玩行程，几乎每个阿尔加维海岸线游船都包括这个景点，同时还可进入观赏一些其它同样令人惊叹的洞穴。（注意要选择包括小船进入洞穴的行程，有的大型船只行程不包括进入洞穴，尽量避坑）

3. 最佳玩法是参加皮筏艇团，自己划皮筏艇进入洞穴。

许多景点只有亲自见到才能感叹它的精彩，同样的，也有一些景点只有到了才会发现“照骗”的由来。这个洞穴是我一直向往的拍摄机位，不过鉴于没法自由进入，加上如今的游客众多，乘船游已经限制不可下船上沙滩拍照了，所以只能在船上随手拍几张了……有点可惜，也为此次扫海之旅留下了些许遗憾。

大船无法进入洞穴，经过约1个多小时的航程，可以更换小船继续参观，而海盗船就这样静静地在海面上等待着我们的归来。

浪漫之都：奥比都斯

这是我第三次来到 里斯本 ，却是我第一次到访这座中世纪的古城。奥比都斯位于 里斯本 市区以北约90公里处，被誉为“婚礼小镇”，是 葡萄牙
人们心中最浪漫的结婚圣地。

自驾至此，小镇下方有一片广阔的空地，作为收费的停车场，远远地就能看到停车场一旁保存完好的巨大 罗马 式水渠。

AFutVABPWqc7QRso160.jpg?imageMogr2%2Fthumbnail%2F1360x%2Fstrip%2Fquality%2F90)

奥比都斯在 葡萄牙 被誉为婚礼小镇，据传有许多 葡萄牙 人来此举办婚礼，造就了小镇浪漫之都的美名的原因则是当年十一岁的 葡萄牙
国王阿方索五世在此与比他还小二岁的新娘举行的盛大婚礼。历史上的盛世早已淹没在时光的长河当中，但当年这一对金童玉女的白色婚礼却又如此深入人心，渐渐地，小镇不断吸引着无数情侣来此拍婚纱照、举办婚礼，最终成为了如今人们所熟知的婚礼小镇。

小镇的历史十分悠久，早在公元前三百年就已有人在此居住，摩尔人统治时期更是修建了大量的水渠和城墙，奠定了这里的建筑风格，这些古老的建筑至今仍完好的保存着。13世纪，
葡萄牙 国王从摩尔人手中夺回了这个村庄，并 大方 地将她作为结婚礼物送给了他深爱的妻子伊莎贝尔。直到1834年，奥比都斯一直都是王后的私人领地。

如今的奥比都斯，始终保持着当年13、14世纪时期的建筑风格，古老高耸的城墙，浪漫蜿蜒的鹅卵石小道，清一色的白墙红瓦，斑驳的门框和泛青的石阶，这一道道古朴简洁的风貌使得这座小镇成为
葡萄牙 最浪漫的所在。

![](https://n2-q.mafengwo.net/s14/M00/30/50/wKgE2l1koxWAYtQaABgMHgRD-

== 古城墙 ==

来到奥比都斯，不可错过的是围绕全程的古老城墙。这座已然屹立千年的古城墙，从8世纪的穆斯林人统治时期初见雏形，尔后经历了时代的交替，渐渐融合了哥特式、曼努埃尔式和巴
洛克 等风格。高高的城墙包围了整座小城，城墙上部只有一肩多宽，一边是墙垛，一边是一丈多高的崖壁，没有任何护栏，行走起来，还是十分惊险的。

不幸的是，在1755年 里斯本 大地震和1808年 法国 入侵战争中，奥比都斯古城受到了破坏，如今的城堡和城墙是20世纪按照中世纪风格重新修建的。

从此处可以购票登上城墙参观，环绕城墙一周约办个小时，漫步其上，可以俯瞰整座小镇的全景。

孔德·德· 卡斯 特罗博物馆，是一栋建于19世纪晚期的建筑，作为奥内尔的官邸，融合了西 方城

外面的蓝色瓷砖墙颇为震撼，又属于 葡萄牙 风格的一大特色，因而成了许多小姑娘拍照的地方。

== 圣玛尔塔 灯塔 及博物馆 ==

坐落在 卡斯凯什 海角上的一座 灯塔 ，始建于1868年，高20米，墙体呈蓝、白相间。如今 灯塔 内部已改造成一个小型的 灯塔 博物馆，在这里，可以了解
葡萄牙 的航海史上举足轻重的 灯塔 故事以及全境内各大著名 灯塔 。

最后来几张远眺 卡斯凯什 城市与海滩的景观，远方那栋庄园城堡建筑称为Palácio Seixas，古老的建筑如今已被改造成一部分的博物馆加上一家高档的餐厅。

== 佩纳宫 ==

坐落于 辛特拉 山巅的佩纳宫，是 辛特拉 旅游宣传的象征，还是 葡萄牙 LP书籍的封面图。这一座浪漫主义宫殿是 葡萄牙 七大奇迹之一，作为 葡萄牙
皇帝的离宫，它兼具了文艺复兴风格、哥特式风格、曼努埃尔风格等各种风格和文化艺术，完美地将各种艺术融合成这一座华丽、耀眼的巨大宫殿，被公认为世界最美丽的十大古堡之一。

红色、黄色的外墙形成了这座宫殿的主色调，拱形门柱、圆顶塔楼形成了宫殿的主要结构，这座宫殿造型奇特而又色彩鲜明，每一个细节都经历了精雕细琢。

佩纳宫参观分成宫殿和花园两部分，也许是看过了 欧洲 许多宫殿，基本都 大同
小异，因此对于内部宫殿不感冒的朋友可以单买花园的门票，同样也能近距离参观佩纳宫的外观建筑。

PS：佩纳宫内部宫殿不允许拍照，外部花园可以拍照

== 摩尔人城堡 ==

== 雷加莱拉庄园 ==

相信有很多人因为那神秘幽长的旋转石梯而种草，也相信很多人来到此处会与我一样认为这里是全 辛特拉 最值得游玩的景点。

雷加莱拉庄园，扩建于1898-1912年，是一座曼努埃尔风格的夏宫。 庄园原属于 波尔图 富商雷加莱男爵，1892年蒙泰罗以25000葡币买下了它，然后邀请
意大利 的建筑师Luigi Manini协助设计，设计了这片4公顷的庄园，如今已被联合国教科文组织列为世界文化遗产，更是 辛特拉 的主要旅游景点之一。

雷加莱拉宫是庄园的主体建筑，进入入口即可看见宫殿的全景，共分五层，包括会客厅、餐厅、主仆卧室、音乐厅、更衣室等一应俱全，主人卧室的窗户正对 辛特拉
山，宫殿的阳台上方雕刻着但丁《神曲》的故事，值得驻足欣赏。

庄园内就像是一个巨大的豪华花园，有众多的湖泊、洞穴、水井、长椅，还有许多精美建筑，其中最吸引人的便是它的地下迷宫，经历过一段10多米黑暗中的岩洞，最终抵达这个深达27米的涸井，周围是双重环绕的螺旋楼梯，站在井底仰望天际，很是震撼。

庄园依山而建，因此高高低低被分成几个层次，这其中则有许多花园、瀑布、长廊，漫步其间，就仿佛置身于一个奇幻的世界。

门票：成人€8，5岁以下儿童及65-79岁老年人€4

交通：乘坐435路公交车至Quinta da Regaleira站下车或从 辛特拉 镇中心步行20分钟

七丘之城：里斯本

在小山村小镇里浪了近一周时间，终于回到了 葡萄牙 首都 里斯本 ，也算是重回大都市了。作为 葡萄牙 的首都，全境最重要的港口城市， 里斯本
拥有着她自己的繁华，同时又是 欧洲 物价最低的 大城 市之一。

里斯本 建立在高低起伏的七座小山丘上，因此又被称为“七丘之城”。早在新石器时代，这里已经有伊比利亚人居住，尔后经历了 罗马
人、摩尔人的统治，如今城市里依然保留有不少古老风格的建筑遗址。直至12世纪， 葡萄牙
国王阿方索一世从摩尔人手中夺回了这片土地，并于1256年定都于此，至此， 里斯本 依靠着地理优势，逐渐发展成重要的港口城市。15世纪起的大航海时代又为
里斯本 带来了取之不尽的财富。

历史上的兴盛总会伴随着没落，1755年的巨大灾难： 里斯本 大地震，近乎夺走了这里的一切繁华景象。地震加上随之而来的海啸破坏了 里斯本
大部分建筑设施，曾经的繁华场面不再， 里斯本 的经济也渐渐走向萧条。虽然后来在 里斯本
人的共同努力下重新规划修建了这座城市，被毁坏的房屋和遗址也得到了修整和扩建。如今的 里斯本
老城区已然又是一片欣欣向荣的景象，但如果你仔细发现，还是能找到许多地震时留下的遗迹和故事。

既然是 葡萄牙 的首都， 里斯本 必然是全 葡萄牙
境内最大的城市，值得游览的景点无数，大部分则集中在下城区、贝伦区和世博区，安排上3-4天时间，以放松的心情去游览各个区域的景点是来到 里斯本
游玩的最佳方法。那么，就让我们从最负盛名的贝伦区说起吧。

== 大耶稣圣像 ==

曾经看到过这么一句话：“如果上帝可以让 葡萄牙 免于战火摧残，就在 里斯本 对岸建造一座彰显耶稣圣心的雕像。”也不知这座彰显圣心的雕像，究竟是 里斯本
抄袭了里约的，还是里约抄袭了宗主国的，总之这里还是很值得一去的。

大耶稣像其实位于贝伦区对岸的阿尔玛达地区，因其高大的身影早已成为 里斯本 城市的地标性建筑之一。

耶稣像高达110米，雕像处是一个观景平台，可乘坐电梯上去（€5），其底座底层有一个小型的祈祷室，一旁即是通往顶层观景平台的直达电梯。

== 4月25日大桥 ==

横跨特茹河的4月25日大桥，全长2277.64米。这座大桥与 美国 旧金山 的 金门
大桥外形十分相似，据说还是由同一家公司承建。大桥建成之际，是当时世界上第五大悬索桥。

== 贝伦塔 ==

再度横跨4月25日大桥，来到位于 葡萄牙 西南沿海的贝伦区，这里有贝伦区最著名的景点：贝伦塔。作为世界文化遗产之一的贝伦塔，在 里斯本
的历史上有着极其重要的历史战略意义，同时它还被评为 葡萄牙 七大奇迹之一。

一大早来到此处，贝伦塔周边已经有不少游客，三三两两地坐在周边草坪和台阶上，而贝伦塔前则已经开始大排长龙，等候进入参观。

贝伦塔始建于曼努埃尔一世时期的1514年到1520年间，最早用于防御贝伦区的港口，如今内部还能看到不少当年保存下来的炮筒还有一些16世纪的文物。后来的贝伦塔逐渐转型成为一座
灯塔 ，是当时 葡萄牙 航海家们探索未知大洋的起点。再后来，这里变成了关押要犯的监狱，进入塔内可以感受到这座地牢的阴森气息。

塔身四周有好几个瞭望窗和阳台，虽然历史悠久，但石块筑成的墙面仍然保存完好，当年的精美工艺至今清晰可见。进入塔内需通过一段木桥，涨潮时的贝伦塔仿佛飘荡于海面之上：

== 大发现纪念碑 ==

如果说贝伦塔是 里斯本 古老的地标，那么大发现纪念碑就是 里斯本
新时代的地标，从贝伦塔走到大发现纪念碑只有几步之遥，沿途是美丽的海域和明媚的阳光，还有一条长长的步行道路。

大发现纪念碑是恩里克王子逝世500周年而建造的，外形酷似一只扬帆起航的帆船，纪念碑的最前端由恩里克王子领头，他手里握着一个小型的 葡萄牙
快帆，纪念碑上还雕刻着达伽马、佩德罗·卡布拉尔、麦哲伦等等在 葡萄牙 大航海时代做出杰出贡献的英雄们。

纪念碑内部有一个介绍纪念碑历史故事的小展厅，可以买票登上二层的露台，从另一个角度欣赏4月25日大桥和特茹河畔美景。

门票：进入纪念碑才需要门票，成人€5，持青年卡者及65岁以上老年人€2.5

== 热罗 尼姆 修道院 ==

里斯本 虽然拥有众多的历史古迹，但鲜少有气势恢宏的大型建筑，热罗 尼姆 修道院算得上是 里斯本
最为壮观的建筑之一了。修道院与大发现纪念碑仅一条马路相隔，和贝伦塔一 样均属于典型的曼努埃尔式风格建筑，是哥特式和文艺复兴式的完美结合体，也是 葡萄牙
七大奇迹之一，1983年与贝伦塔一起被列为世界文化遗产。

修道院建立于 葡萄牙
的全盛时期，于1502年动工，历时100年左右建成，整座修道院由打磨得光滑整整齐齐的白色花岗岩堆砌而成，其门面长达300米，外观之宏伟，内部之华丽可谓是全
葡萄牙 之最。1755年的 里斯本 大地震，整个 里斯本 几乎毁于一旦，唯有修道院屹立不倒，为这座建筑增添了一层神秘色彩。

从空中看去，修道院的整体结构像一个十字，内部则分为教堂和修道院两部分，一般先从教堂参观起，许多 葡萄牙 历史上的重要人物长眠于此，其中包括卡 蒙斯
、达伽玛等。

PS：如今的修道院已成为游客重点参观目的地，等待入场的游人排队无比之长，建议提前在网上购买好门票，按时入场，省去排队时间。此行我因为预估失误，见排队太长了，没有进入，只剩下几年前第一次来的时候拍的一些渣图了。

离开教堂，比较值得参观的是修道院的中庭，虽然看起来比较空旷，但从上俯瞰，还是能发现庭院八角形结构。庭院中最为迷人的是环绕着的拱廊，精美的石雕遍布四周，长长的走廊在阳光下留下了许多迷人的光影。一定要选在阳光明媚的日子来此，否则无法真正感受这里光影变化的魅力。

== 乘坐 里斯本 叮叮车感受历史的痕迹 ==

每个来到 里斯本 的游客，一定都听说过这里拥有着 欧洲 最古老的电车，这些造型复古的电车已然成为 里斯本
街头的一道亮丽风景线，无论大小街道，总能找到电车的身影和那传统的叮叮声。因为 里斯本
高低不平的地势，电车和升降机成为当年最为风靡的城市交通工具，现存的几个著名升降机更是坐落于老城中心重要的街道上，乘坐一回当地的复古电车也便成了游人们来到
里斯本 必体验的项目之一。

接下来将为大家介绍几个最有名的线路。

1. 28路电车

里斯本 至今保留的最有名的电车线路，所有的28路电车均是黄色的，所经之处几乎环绕了 里斯本 老城的重要景点，因此，这两黄色的小电车在不知不觉中也便成了
里斯本 城市的标志之一。

里斯本 电车公司创立于1872年，是 葡萄牙 最早经营公共交通运营的公司，如今仅存5条线路仍在运营。假如你想要快速、简洁的了解 里斯本
这座城市，那么，坐上28路电车走一圈。古老的建筑，老城里的大小街道，庄严的 里斯本
大教堂，宏伟的圣乔治城堡，这一切的元素与风景，均能从28路电车的车窗外一一欣赏。

升降机坐落在 里斯本 古老的老城区，这里是 里斯本 古迹和重要建筑的聚集地，起点从山丘下方的中兴斗士广场(Praça dos

== 圣佩德罗· 阿尔坎塔拉 观景台 ==

老城区中我个人认为比较好的观景台，地处荣耀升降机上方，无论白天还是黑夜，都是眺望圣若热城堡和阿尔法玛区的好地方。我们住的酒店就在荣耀升降机边上，傍晚时分来到此处，已经有不少游客在此等候黄昏。熙熙攘攘的人们，有的静坐在长椅上，有的拿着啤酒聊天，有的则静静地站在栏杆边上看着眼前的美景。

平台实际上以一片建于19世纪的花园，中间有一个没有水的小喷泉，周边几个小商贩售卖手工艺品和饮料零食，坐在这里等待夕阳，身旁还有街头艺人们弹奏的悠扬乐曲。

== 阿尔法玛老城区 ==

说完了老城区和贝伦区，接下来让我们来看看 里斯本
最古老的街区，阿尔法玛区。这片古老的街区建立在一座小山丘上，各式各样葡式外观鲜艳而亮丽的建筑坐落在起伏的山坡上，那宛如迷宫的弯曲起伏的窄路旁，隐藏着许多当地的古老风情。沿着主干道的山坡向上，途经
里斯本
主座教堂和若干可以俯瞰阿尔法玛区及太加斯河的瞭望台，上文提到的圣若热城堡同样坐落于此区。阿尔法玛的街道是鹅卵石铺就的，错综复杂，走在期间就像是行走在迷宫当中。但如果不想走太多的路，那么可以选择乘坐28号电车行经阿尔法玛区，在车厢内欣赏当地的美丽。

作为 里斯本
最古老的区域，这里的建筑都有着不菲的历史价值，狭小的街道甚至无法驱车进入。遍布的古老建筑，有些已几经翻新，有些依然破旧不堪，这里散落着许多精致的小店，同时一些著名的法多音乐场馆也在这里，在欣赏完黄昏之后，不妨选择一家小馆，品尝葡式美食，听一场法朵音乐，喝一杯小酒，放纵一下自我。

== 里斯本 主座教堂 ==

坐落在阿尔法玛区上山的主干道上，也是28路电车途经的重要景点，曾经是摩尔人统治时期最为繁华的街区。教堂前身是一座清真寺，后来被阿方索国王下令改建，成为全
里斯本 最古老的教堂。后来的大教堂在经历多次地震中，又几经损坏和重建，最终 新城 了如今展示在世人眼前的大教堂。

== 观景台 ==

从主座教堂向上继续行走几分钟就可抵达著名的观景台，这里是阿尔法玛区一个重要的观景平台，许多年轻人和游客来到此处欣赏阿尔法玛区的美景，眼前的一切是一片连绵的白墙红瓦，远方蔚蓝的太加斯河和蓝天白云相映，不绝于耳的音乐声，人们轻声的攀谈声，还有平台一旁小酒吧不断传出的杯子碰撞声，加上眼前的美景，都令人心旷神怡，不由得想要停下脚步，等待夜幕的降临。

许多年轻人和游客坐在平台的围墙上聊天，画面十分宁静和谐。

里斯本 的建筑多为白墙配上色彩鲜明的门窗，加上门前的一颗老树，也是一副很清新亮丽的画作。

从观景台继续向上，回来到另一个高地平台，这里不是著名的观景平台，确实无比热闹的酒吧聚集区。

== 里斯本 博物馆推荐 ==

1. 瓷砖博物馆

这是我在 葡萄牙 参观过最具特色的博物馆，相信每一个来此的人都不会失望的。在 葡萄牙
的历史上，从摩尔人时代就开始制作彩绘瓷砖，使用瓷砖装饰曾是财富、权利和身份的象征，只用于重要建筑上，后来才渐渐流入民间。在 葡萄牙
，你一定能够从不同的瓷砖发现各种历史故事，这些瓷砖上的绘画多数和宗教、历史相关，当然也有一些风景、几何、人文民俗的图案，在这座博物馆里珍藏了众多的瓷砖，值得慢慢去欣赏这些精心绘制的瓷砖上的图案。

hABNGClmVLmA945.jpg?imageMogr2%2Fthumbnail%2F1360x%2Fstrip%2Fquality%2F90)

就连厕所的标志都是瓷砖做成的……

2. 航海博物馆

博物馆紧邻热罗 尼姆
修道院，也许是修道院的闻名掩盖了这座博物馆的名气，大多数人都在排队等待进入修道院，使得这座博物馆显得很安静。展馆内主要展示了从中世纪之前的小木船，到具有多层火炮远航帆船，最后到轮船、核潜艇等各种船模。此外，博物馆里还讲述了大航海时代的各种历史故事，包括一些详尽的航海线路图和当年的造船技术发展。这是可以超出预期的博物馆，如果你是模型爱好者，或者航海爱好者，那么来到此处必然不虚此行。

门票：€6.5

3. 马车博物馆

马车博物馆（Museu Nacional dos Coches），主要展示了不同年代的马车，从简陋的平民马车到豪华的国王座驾，这里应有尽有，也是全 里斯本
乃至全 葡萄牙 参观人数最多的博物馆之一。博物馆原址是皇家骑术学院，至今收藏了从16世纪到19世纪的各种马车，完整的重现了马车的辉煌发展史。 葡萄牙
作为当年的世界强国，其皇室、贵族所使用的马车工艺可谓是美轮美奂，精美至极，看到这里面大量的皇家马车，绝对会为上面精致的木雕工艺和奢华的设计所惊叹。

门票：单马车博物馆€8，皇家马厩€4；马车博物馆+皇家马厩联票€10；马车博物馆+阿茹达皇宫联票€12

马车博物馆和皇家马厩不在同一栋建筑里，即所谓新馆和旧馆，现代化的展馆较大，是新的马车博物馆，里面的展品也比较多，可以选择看这一个就够了。

== 保险 ==

出国旅游，无论是去哪，都有购买旅游保险的习惯，并且申请签证时一般也会有要求附上保险（且满足以下3个条件）：

① 医疗保障保额不少于3万欧元（相当于30万人民币）；

② 必须包括国际紧急救援服务、送返回国就医、紧急医疗急救以及紧急住院治疗；

③ 保险期必须覆盖全部境外行程。

== 网络 ==

没有买电话卡的，直接电信开通的国际漫游，用于接打电话。（注意，国内电话卡开通国际漫游用不了谷歌地图！）

网络方面全程使用的漫游宝移动WiFi，几乎已经养成每次出国必备的移动WiFi，小巧却可以连续使用一整天。各大平台都可以预定，在机场取还，在城市里信号很好，即使自驾到那些小镇里也有信号的，对于全程用谷歌地图导航的我来说简直是救命了。

== 货币与信用卡 ==

现金：在 欧洲 直接使用现金的概率比较小， 大城
市里大大小小的商店全都可以使用信用卡。出发前准备600-800欧元现金在身上差不多，尽量不要带大面额的现金，容易被偷，而且小店里也不收100以上的。

信用卡： 欧洲 旅游信用卡是必备品，建议出国前先在国内申请好一张外币单标信用卡（双标卡被盗刷风险高于单标卡）， 欧洲
主要用的是VISA和MasterCard两种，JCB是 日本 用得多，AMEX则是 美国 用得多。

PS：有想要了解各行信用卡权益和适合的信用卡，请加 北斗 微信私聊，作为专业羊毛党给与最专业的建议。

银联： 中国 银联卡分储蓄卡和信用卡两种

储蓄卡很简单，大多数国外POSS机是用不了的，不过在当地ATM用银联卡直接取现的汇率绝对比去货币兑换点换钱来的值，所以我一定常带一张储蓄卡一杯不时之需。

银联单标信用卡，则有可能遇上一些地方使用不了，虽然如今银联已越来越国际化，但还是仅限于 中国 人使用， 大城
市、大商场基本可以用银联，甚至有促销，但是我们去 葡萄牙 ，离开了 里斯本 机场就基本用不了银联了。

== 小费 ==

在 欧洲 是非强制给小费的，一般高档点的餐厅小费在结账时已经包含在里面了，如果不包含的话则是自愿给的，给多少也很随意，毕竟 欧洲
服务员不是靠小费为主要收入的。

== 厕所 ==

专门拿出来说是因为在 欧洲
几乎看不到公共厕所，不同于国内的肯德基、麦当劳，这些快餐餐厅的洗手间经常都是需要密码的，也就是非本店顾客不可使用，一些公共区域的洗手间也都是收费的，一般0.5-1欧之间。以下有几个找洗手间的好地方：

1. 收费景点或者博物馆内有免费洗手间；

2. 大型购物超市、商场有免费洗手间；

3. 机场洗手间免费，小镇等地的火车站洗手间有可能收费；

4. 所有餐厅只要消费了，就厚着脸皮使用吧，特别是街边的小咖啡馆，往往一杯1-2欧的咖啡也跟找公共厕所的费用一样了……

== 当地治安 ==

葡萄牙 作为曾经的海上霸主，如今已然没落了，相比于 英国 的金融繁华、 法国 的浪漫文艺， 葡萄牙 的小镇显得萧条了许多，难民也相对少了一些。离开 里斯本
到南部的海岸线，整体治安很好，因为基本没人。 里斯本 则是游客聚集区，移民、难民聚集区，市中心小偷还是有的，一定要注意好自己的随身物品。

== 当地特产 ==

葡萄牙 的大型购物中心基本都集中在 里斯本 ，如果想要购买名品等只需要最后在 里斯本
商场血拼即可，然而这些东西都不是我喜欢的，对于名牌没（买）兴（不）趣（起）。每次出行，本人都比较热衷于购买当地的小工艺品特产，希望将来可以将自己的家打造成一个集世界各地特产、工艺品的小型展馆。

这里说说 葡萄牙 适合购买的工艺品和手信：

1. 软木制品： 葡萄牙 最重要的工艺品，也是全世界最大的软木制品出口国，包括软木包、笔记本、明信片、甚至是皮带、项链等，在大大小小的纪念品商店均能找到；

2. 瓷砖：特别是蓝色瓷砖，最早由阿拉伯人传入 葡萄牙 的瓷砖制品如今已成为 葡萄牙 城市大街小巷必备的装饰品；

3. 印花布：小镇上当地居民都会出售很多手工制作的印花布， 比如 手帕、桌布等，小哥哥们可以买一条小手帕送给小姐姐以表爱意；

4. 红酒： 葡萄牙 的红酒虽然不及 法国 的风靡，但是当地特产的波特酒、 马德拉 酒也是值得体验的；

5. 蜂蜜： 葡萄牙 南部小镇的特产， 法鲁 地区尤为突出，在阿尔加维很多地方都能找到卖蜂蜜的小店；

6. 鸡🐔：大大小小形态不一的木质葡国鸡，或者彩色葡国鸡冰箱贴也是许多游客必买的纪念品，好吧，出于情怀买一个还行。

租车攻略

作为摄影发烧友，无论去哪我都会选择租车自驾旅行，多年来的租租车忠实用户，不为别的，只因租租车确是我至今用得最习惯的国外租车平台，价格实惠而且服务十分完善，无论在哪租车，基本上大部分租车公司的车子都能搜索到，可以直接对比价格，也很容易筛选需要的车型，全中文平台，搜索起来一目了然。

租车流程：

1. 在租租车网页或APP选好时间和取还车地点，通过筛选功能选择手动/自动、车型、座位数等。

2. 货比几家，选择心仪的车型下单，下单时注意租车条款，包括租该车的最低持有驾照年限、额外驾驶人费用、里程限制、是否满油取还。

3. 填写好自己的个人信息，提交订单后驾驶员信息不可更改，只能取消订单重新预定。

注意事项：

1.
一定要买保险！而且要买全额险那种。不要以为自己车技好就没事，遇上一次修车费用哭死人。买保险我一向都是租车的时候直接在租租车订单里选择了全额险，这也是我为什么喜欢使用租租车的原因，因为自带的保险是国内保险公司的，如果出现刮擦需要理赔时，在还车先自费维修费用，回国后APP上上传所有材料截图和信用卡扣款记录，很快保险赔款就能到账了，而且全程有中文客服跟踪，放心省事。

2.
取车时需准备好：护照、驾照、本人名下的外币信用卡，有驾照公证或者租租车小白本也可以带上，信用卡一定要带带芯片的外币单标卡，双标信用卡很多时候无法使用；

3. 拿到车时注意检查车身上的划痕，因为岛上多为旧车，小剐小蹭很多，认真拍照让工作人员记好；

4. 和 欧洲 大部分国家一样， 葡萄牙 的红绿灯不多，十字路口多数是环岛，切记环岛内车辆先行；

5. 在 欧洲 开车一定要 礼让 行人，另外有红绿灯的十字路口，不可直接右转，需等绿灯后才能右转；

6. 在租租车平台租车的时候可以免费办理国际驾照翻译认证件，现在还加了法语、德语、 西班牙 语等宣誓认证，建议租车同时办一个认证件，方便使用。

== 加油 ==

欧洲 的加油站多是自助加油的，一定要确认好自己的车子加的是什么油，选择自助加油站的好处是油费肯定比人工的便宜，自助加油流程如下：

1. 插入自己的信用

2. 授权信用卡并选择加油类型

3. 有的需要选择加多少钱或者加满油

4. 取出信用卡并拿下对应的油枪

5. 感受到油泵启动的震动后自己把油枪放入加油即可

6. 达到指定的费用或加满油后油枪会自行停止

7. 放回油枪，有的加油站还需要再插入信用卡确认扣款

葡萄牙 的油价属于比较贵的，约1.3-1.5€/L。在 葡萄牙 高速行驶，或者进入乡村里面，加油站的数量相对较少，所以尽量保持油箱有一半以上的油。
