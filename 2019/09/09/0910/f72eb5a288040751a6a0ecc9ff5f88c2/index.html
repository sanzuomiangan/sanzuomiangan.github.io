槟城这座南洋之城，原来一面是城一面是海

发表于 2019-09-09 |

前言

说起槟城这个岛城，我第一时间会想起壁画街和各种各样的美食，毕竟过去无数次在网上看见槟城的攻略里，也大多是这两方面的推荐，印象里的壁画街就真的是一条街，印象里这里的美食种类多到每天从早吃到晚不重样也不觉得过分。

其实这次计划去槟城，实在是太意外了，我和猫叔从订机票做行程到出发，真的就是一周内完成的事情，幸亏槟城一年四季都是“夏天”，打包好裙子就可以出发了。出发之前，为了想更多了解下这个目的地，我还特意找上曾经在槟城留学的小姐妹，让她给我出点主意，没想到她却颠覆了我对槟城的印象，她跟我说：“槟城也是个岛，也有属于自己的海，而且也非常值得去看看，那附近也有很多好玩的地方，都是本地人居多，因为马来西亚有太多有名气的海岛了，槟城的海滨名气自然被淹没了，游客大部分只知道乔治市。”

槟岛的另一面，是骑马看日落的海滨

抵达槟城后，我们直接打Gap到槟岛的北部，这里有一个耳熟能详的名字—巴都丁宜，一路绕着盘山公路，经过槟城最繁华的城市，也经过槟城最深山里的弯道，边走边看不知不觉就到了。我们选了一家在网上风评很高的海滨酒店Park
royal，据说是海滨最好配套最完整的酒店了，办理入住手续是在大堂旁边的沙发凳上，这种方式也蛮度假的。

我的房间在转角位置，活动空间比较大，特别适合猫叔这种吨位的人了，开心了老半天。

放下行李在酒店范围里走走逛逛，这个酒店也太网红了吧，有很多摆设都特别适合拍照，配套设施也很齐全，而且如果你是有心人，真的会在酒店里发现很多“艺术品”呢！在一层竟然还有一个“水上世界”！大人小孩都爱玩～

霸占一个小木墩，坐下来喝杯东西，安安静静等着日落。我真的等了这个“小木墩”很久才成功霸占到！赶紧拍几张日落的照片。

和其他海滨一样的是，在这里你可以体验到各种不同类型的水上设施，沙滩上有不少玩乐设施的公司，大家可以自个儿去问问价格。这次来没打算下水的我，选了另一种很特别的方式，日落时间在沙滩边会有很多人骑着马出来溜达，如果感兴趣可以去砍价，大约40马币可以溜达沙滩一圈，各种拍大片！

在巴都丁宜沙滩边有一家当地无人不知的网红餐厅Bora Bora，我们在入住Check in的时候服务生就强烈安利我们，因为就开走酒店隔壁，步行也就5分钟距离

漫步在沙滩边，你会发现整个沙滩都没有摆到外面的凳子或者桌子，唯独Bora Bora，所以特别多人会在这里看着日落吃晚餐。

这家Bora Bora风格太海岛了，室内是小酒吧，室外是餐厅居多，我来的时候户外沙滩座位剩下最后一张桌子，赶紧坐下来霸个好位置欣赏日落去

吃饱喝足之后，门外主干道就有一条最短的夜市街，道路两边都是各种小店，还有不少大排档，记得买一个榴莲吃！百元内吃到饱了！

世界上最小的森林公园，上山下海什么都齐了

槟城的国家森林公园，是世界少最小的一个，门口就小小的一个，进去之前需要在门口填写好你的护照号/身份证信息，公园虽小却很负责任，为了确保每天进园的人数，保证多少人进去多少人出去，所以有了这一个步骤。

来这个森林公园里玩法还挺多的，里头有两个沙滩，其中一个monkey
beach有很多猴子，是唯一可以游泳的地方，一般如果可以停留一天时间，建议可以进去徒步，沿途会经过非常多超美的点，而且还有一些探险类的设施，最后走到海龟哺育中心去。如果时间不足的话也可以选择海路，直接坐船去某一个点体验，比如我选择了去海龟哺育中心，距离码头最远。

乘船途中师傅也会边走边介绍沿途的特色，这块石头也太像兔子了吧！！

海龟哺育中心在最远的无人沙滩上，整个沙滩边就我们一艘船，踩着细软的沙子漫步，还蛮舒服的，比较适合坐下发发呆。

这个沙滩上只有一个海龟哺育中心，门口也是小小的，脱鞋子进去就是一个超级小的“泳池”里头刚刚出生2天的海龟bb出来打招呼，简直萌化了！

里头有很多关于海龟保护的咨询和信息，还有几只大海龟之前受了伤被送过来治疗，慢慢康复的小龟龟们不久之后又会被送回大海去了。

当地有一个很受欢迎的团就是过来看海龟下蛋的，走的时候哺育员还带我们去围观门外已经圈起来的“下蛋锅”，期待不久之后新生的海龟bb们吧～

误入INS同款蝴蝶园，简直就是一个巨大的科普课堂

在国家森林公园不远，有一个同样是比较少游客知道的蝴蝶公园，成年人27马币一位，进去可以逛一个下午完全没问题而且不觉得乏味，大人小孩都超级无敌适合！

在门口买票的时候，工作人员会推荐我扫一个二维码，里头有很多不同主题的互动环节，可以和“小伙伴”们亲密接触呢！还有什么蝴蝶放飞时间，看着就觉得很赞～

进去以后顺着参观指引走，会经过各种不同的区域，首先进去的是一个阴凉的小山洞，里头住的全都是毒类小动物，看着样子人畜无害，没想到！你萌竟然有毒！！最好笑的是在这个小房间里，我和猫叔一直上演“找找我在哪”的游戏，有些小昆虫实在是太难找到了吧！

往外走到了蝴蝶公园重头戏！整个玻璃屋里都是各种各样的蝴蝶！围着你翩翩起舞呢！园中间有一颗巨大的树，是整个蝴蝶公园的中心树，离远看怎么有种新加坡视角呢？

“Hi大家好，我是这个园区最可爱最受欢迎的宝宝”这三只活宝住在一玻璃屋里，运气好的话你还能看到它们冲你挥手呢！

目前蝴蝶园里拥有120多种蝴蝶品种，共四五千只，还有部分其它种类的昆虫和植物，慢慢走慢慢看，再听一下讲解，真的科普不少。

走热了走累了的时候，不知不觉就进入了一个空调房，这里头是一个巨大的科普课堂，有很多关于蝴蝶和其他昆虫的科普咨询，还能亲眼瞧见小蝴蝶诞生呢！里头也有很多位置可以体验互动！炒鸡棒！

喂喂喂喂，你们在干嘛！！

准备离开之前赶上了另一场互动环节，就是我最期待的放飞蝴蝶之旅了！每天新出生的蝴蝶宝宝们，会被带到这里，由游客一起放飞，既有意义又新奇！我的这只蝴蝶宝宝好像特别粘人，还做起我的新饰品来了，咳咳大家好请看我的搭配。

被列为世界文化遗产的古董酒店E&O

在海边住了两天，我和猫叔决定搬家到繁华热闹的乔治城，感受感受城里的魅力，选择酒店简直就有困难症，乔治城里有太多有特色的好酒店了，最后我们决定选择这家在乔治城里无人不知的古董酒店东家酒店。

这次我们住的是新翼，被列为世界文化遗产的旧翼正在升级改造中，有点小遗憾没有成功住上，但新翼这边也足够让我们惊喜了，整个新翼的所有房间都是看海的，而且配套设施更加新，一点也不逊色于旧翼。

我们住的房间有一个超级巨大看海的浴缸，马上来一张照片～

东家酒店是很典型的英式建筑，这么多年来都这么受欢迎，酒店里还有一个历史回顾区域，很多老顾客会把自己在酒店里的体验照送给酒店，酒店把这些故事记录下来陈列给大家看，实在是太走心了。

我和猫叔有点小偷懒，想“叹”一下东家酒店最有名的下午茶，他家下午茶我真的种草很久了，虽然现在没办法在旧翼酒店里感受到那种在“古董”里下午茶的感觉，但还是可以在新翼这边感受下他家完美的出品的。

我们点了一份双人下午茶套餐，甜的咸的都有，还有两份茶饮，吃得我和猫摸着肚子走～

槟城虽小，网红博物馆却超级多

槟城乔治市面积没有很大，最奇妙的是我在做攻略的时候发现，除了壁画街，这里竟然隐藏着这么多博物馆？各种各样的博物馆大大小小什么主题都有，每一家都各具特色，吸引了不少人打卡。我这次只选了两家去试试，第一家就是食物狂想馆了。

食物狂想馆给我的印象太深刻了，还没进门就有一碗超巨大超逼真的煎蕊，里头还有各种比例和主题的美食

“哇！！猫！你见过这么大一碗的红豆冰吗？”

“这么一大盘的美食？感觉好好吃啊！“

“放下这榴莲让我来！“

两个吃货逛这个馆，所有的对话就离不开吃吃吃吃吃了，让人脑壳疼。

不仅好看好“吃”，馆里每一个环节都互动性超强的，可以拍出很特别的照片来，重点是馆里的食物也太逼真的吧，一边了解槟城不同美食的故事文化，写着写着真的有点饿了。

除了食物狂想馆，不远也有一个3D艺术博物馆，整个馆面积不小，按不同主题分区，一进门就是东南亚特色的椰子树林！还有槟城最有名的目的地背景，上去互动一番波，拍出来感觉还蛮特别的。

每个拍摄区域都会有指导，推荐大家最佳拍摄地点和角度，如果不会拍工作人员还会出来帮忙呢！大家需要做的事情就是和我一样放松来边玩边拍就好

我和猫叔在里头玩得不亦乐乎，还搞起创作来，这大小琪琪给几分？

乔治市的那些事儿，原来壁画街不是一条街

漫步在乔治城里，意外发现乔治市的风格也太特别了吧，简直都很适合拍照呀！

城里的包容度非常高，简直风格也大有不同，有印度风、南洋风、英式风、华人风等等，走在大街上不知不觉就走了小半个乔治城了。

在没去槟城之前，我一度以为壁画街就是一条街，整体街从头打卡到尾，当我真的踏上乔治市的时候，我才发现原来这些壁画，分布在乔治城不同位置，有一部分甚至隐藏在小巷子里，真的需要花点心思慢慢去找。

这一幅经典的壁画最好找了，我在找一家当地很出名的网红咖啡店的时候在隔壁看见的，据说周末的时候这里会大排长龙，赶紧趁人少拍一个。

猫叔挚爱的篮球壁画～

这三幅壁画在靠近姓氏桥的位置，比较集中好找些，都很适合互动，但其实这条街里的第一幅壁画其实是小黄人。

拍完壁画隔壁还有一个非常酷的创意园可以去逛逛看，里头有很多手工艺品和设计店，还有一家网红咖啡店！

下面这一幅在恋爱巷的后面，超级大的一幅壁画，对面还有很多学生在这里写生呢！

还有这两幅是我散步的时候无意中发现的，也是靠近恋爱巷不远，觉得蛮特别的赶紧拍下来了。

我和猫叔冲着电影取景地跑去恋爱巷，却发现白日里的恋爱巷没什么特别，一问才知道原来正确打开方式是在晚上。晚上这里摇身一变就变成一条酒吧街了，摊位都会摆出来，还真蛮热闹的。

过马路找吃的还偶遇到最美日落！鸡冻！！

在爱情巷的马路对面，入夜之后是一条宵夜街，各种各样槟城的美食都能吃上，特别推荐一家云吞面店，视频里头放进去了！嘶～想起来有点想吃了

乔治城里的老建筑，都是颜色各异的网红屋

在做攻略的时候，我和猫叔只选了两个看起来比较有意思的老建筑去参观，没想到真的震惊了，这些老建筑一点都不老，而且还很网红，不仅能了解相关的背景和历史，还特别适合拍照！！

这间小蓝屋就是著名的张弼士故居了，这间屋子是东南亚现存的最大的清代中国园林式住宅，据说也是全大马风水最好的宅子，因为隔壁改造成了精品酒店，为了保证住户的私密性，现在每天只有几场的英文解说团，可以带大家进去参观一圈，大家如果要来的话记得看好时间了。

整个宅子里也太有讲究了吧，小到一个雕花装饰，都有风水讲究，据说外墙壁的蓝色，是用一种叫蓝花的花汁加工之后染色的，感觉也太小清新了吧～

娘惹博物馆

交通：侨生博物馆可由槟城市区乘免费旅游巴士抵达，也可步行抵达

门票：成人20马币，儿童免费

开放时间：09:30-17:00 (1月1日-12月31日 周一-周日)

一开始听到参观娘惹博物馆，猫叔是有点抗拒的，结果跟我跑了一趟之后，我俩真的有点懵了，原来娘惹跟想象中完全不是同一回事，娘惹文化竟然这么有意思的。娘惹博物馆的颜色和张弼士故居不同，是一间小青屋，门口就已经非常有气派了，里头简直就是富丽堂皇！土豪的生活！！

蹭了别人的导游讲解才知道娘惹这个词，指的是华人和马来西亚人的混血女儿，如果不是腰缠万贯的还不能称呼娘惹，如果是男孩子的话则会被唤作峇峇。这座融合峇峇娘惹住宅风格的四合院，不只是内部与外观保持完整，在这栋百年古宅内，更是收藏着上千件古董和珍藏品，晶光闪闪的峇峇娘惹文物，金漆屏雕精致典丽！天呐～满满都是黄金的味道，富丽堂皇之余真的随手都能拍大片，也难怪那么多电影喜欢在这里取景拍摄～

往上走二层是峇峇娘惹的房间，里头还有当时闺女出阁的布置，什么礼服、闺阁秀女的妆奁柜、绣花鞋、凤冠霞帔、嫁喜礼饼盒等等，都还完整保留着。

二层的小阳台拍照实在太漂亮了，忍不住拍了一张

这个就是他们家的洗澡池子，够不够高大上！真个馆慢慢逛下来还真能逛个两三小时，主要是太能拍了，我和猫叔完全舍不得离开～

原来所谓的姓氏桥，是每个姓氏一条桥

在乔治城里，有个非常有名的景点目的地叫姓氏桥，每一个到槟城的游客都会来这里打卡，火爆程度不亚于壁画街，一开始我也以为姓氏桥就是一个地方的名字，没想到原来是一片桥的统称。过去华人到了槟城，以海为生，搭起桥板，在上面建造房屋，因为同姓者同住一桥，所以称为姓氏桥。所以传说中的姓氏桥并不是一条桥，而是每个姓氏一条桥。

这一条桥是游客最多的，往外走去会看到一排排渔船，但其实旁边的姓陈桥是最多网红照的机位，来这里的游客并不多，往外走去只有一条木桥延伸到外面，一间铁皮屋，还蛮可爱的。

如果有耐心的话，感觉每一条桥都去看看也不错。

打卡乔治城的网红咖啡店，AWSOME CANTEEN

作为一个经典的咖啡控，每到一个地方我都特别喜欢去找一下当地的精品咖啡店，品尝下他们的出品环境之余，还可以做个探底。从姓氏桥出来，我在INS上搜到一家不远的咖啡店，是一家森林+工厂风的精品咖啡店，赶紧拉上猫叔就跑。

我们来的时候已经接近五点了，所以店里没有很多人，但可以想象出来白天会有多么火爆，这环境也太太太有意思了吧！我们找了个位置坐下来发发呆，点了一杯冰拿铁，随手就拍几张。

在这附近还有很多特色网红咖啡店，对面有一家名声在外的“槟城最长咖啡店”，也很值得去逛逛～

谁说老景点只是走马观花？

在槟城有很多非常经典的景点，很多人都会觉得太传统没什么意思，匆匆过去打个卡就走了，这次我和猫叔其实也去打卡了这些目的地，但却发现了不少非常新鲜的玩法，以后别再说老景点只能走马观花了。

门票：免费，登万宝佛塔2马币

开放时间：09:00-18:00 (1月1日-12月31日 周一-周日)

槟城其实是一个宗教自由的城市，在这里你可以看到各种各样的宗教建筑，我和猫叔准备去打卡升旗山，顺道把隔壁的极乐寺也一起去了，没想到还真的颠覆了我对寺庙的印象。

打Gap来到山脚下，可以慢慢沿着石梯往上走，没走多远经过一个超级美的花园，这里真的是INS网红巨经典的打卡位置。

整个极乐寺虽然是寺庙，但却融合了中国、泰国和缅甸三种不同的建筑风格, 和其他寺庙不同，极乐寺里的建筑五彩缤纷，超级好看！所以也是马来西亚的重要佛教寺庙之一。

这里的最大看点便是高30米，共有7层的万佛宝塔，塔的设计十分奇妙，塔身素白，塔顶为缅甸式，中层为泰国式，而底部为中国式，外形和白蛇传中的雷峰塔相似，塔内每层都摆有大理石制的佛像及全身金箔的佛像，四大金刚和弥勒佛相当引入注目，坐、立、卧姿态各异。

从下面付2元坐缆车到山顶，还有一尊全世界最大的室外观音像，花费了27年时间精心建造，周围由16根大柱围建起来的八角亭遮盖住，现在其实还没完工，已经可以看出她的庄严肃穆了！让人叹为观止！

升旗山

交通：从乔治市乘204号巴士可达升旗山站

门票：外国游客:成人、老年人30马来西亚林吉特/快速通过票80马来西亚林吉特 (1月1日-12月31日 周一-周日)

半票:儿童(4-12岁)

开放时间：06:30-23:00；停止售票时间:21:45 (1月1日-12月31日 周一-周日)

山上的另外一边，大家一定知道，就是著名的升旗山，从山下往上一定要坐这个经典缆车，我和猫叔提早做好功课来的，上山记得坐在车头第一排，下山则要坐在车尾第一排，感受感受过山车视角！

上山逛了一圈，遇到雾霾天起能见度并不高，其实有点小沮丧，拍照也拍不好看，所以我们决定去体验不同的玩法，在山顶广场买了The
Habitat的门票，会有独立的电瓶车把我们载到入口，这里头实在是太大了，有大家平时在网上能看到的玻璃圆顶，也有我们这次花了2小时体验的超级无敌刺激丛林穿越之旅。

对！我们玩的就是丛林穿越！但升旗山的丛林穿越感觉更全面更升级了，全程大约2小时左右，各种各样不同方式的飞行，紧张又刺激，从上往下走的时候，紧张得嗷嗷大叫，还给猫叔取笑了老久了。

从The Habitat出来，记得不要提早往回走，沿着道路继续往前去，大约走个800米左右，会见到一个超级火爆的森林咖啡店Kopi
Hutan，这里除了是个网红咖啡店，还有一个monkey cup的猪笼草植物园，可以买票进去参观的

我们来的时候人比较多，在外面等位置，老板还把他自己的“宠物”蝎子拿出来玩，还非得放到我们手上，吓得我赶紧跑开！

等了大约十分钟，终于等到座位了，在大自然里喝咖啡还是头一回，隔壁小蜜蜂一直缠着我们不放的感觉蛮特别的。

喝完咖啡店里会有车把大家拉到山顶广场去，所以不用担心交通问题～

在乔治市最高的建筑上拍爬楼照同款

从山顶下来，我又跑去另一个高点了，乔治城里还有个地标其实很值得一去，就是光大大厦了。光大大厦是槟城的一座非常知名的地标建筑，有65层高，楼下是槟城市区内最大的购物中心，里头有超级多不同主题的馆可以慢慢玩，如果时间充足买套票最划算了。

我第一站直奔楼顶，坐电梯上去经过第一个观景台，有一种超酷炫的玩法，大胆宝宝的我当然不会错过，在柜台买票可以穿上装备环绕光大大厦外围一圈，各种高难度危险照片随手拍，真的超级酷的！胆小鬼猫叔一开始有点紧张，虽然有保护措施，毕竟还是蛮高的，心里直发毛，后面被我逗乐了也跟着我一起做高难度动作拍照去了～

看我的Pose～胆子大玩啥都嗨

这两张我真的超级喜欢，有木有爬楼视角？

往上走还有一个The Top的露天平台，这儿有一个彩虹玻璃拱桥，也是很挑战勇气的，穿上鞋套走到外面可以摆拍很多酷炫的pose～

隔壁写着“世界上最幸运的凳子“，赶紧坐一下蹭点运气来，后来才知道这张凳子是用马蹄做成的，嘶，疼死我了

从楼顶下来，我拉着猫叔跑去下面的水族馆，这个水族馆和平时看见的有点不同，这个馆把每一个布置做成特色美食的摆设，还真蛮特别的，而且还有一些小互动环节，大人小孩都蛮适合的，反正我这个海的女儿表示超级喜欢！

在槟城，一日六餐餐餐不重样

我不会告诉你们在槟城的短短一周时间，我胖了8斤的！在槟城这些天真的感觉再大的胃都不够用，太多好吃的等着我们来试了，最后一天我和猫叔选择跑去大排档街扫街吃吃吃，什么结婚减肥的事儿已经不存在了，回国再好好减肥吧

经过先点一份全槟城最有名的潮汕煎蕊～

乔治城里有太多大排档街了，潮汕街算是一条，美食街也是一条～我们来到美食街的时候天还没暗下来，就已经坐满了人，在槟城吃华人大排档有一个传统，你坐哪家的凳子，就要在哪家有消费，不然真的会被赶走的（不要问我为什么知道，我是不会告诉你的），我们选好位置就开始扫街了，这家炒粿条排队排到外面，人气超级火爆的，当地人吃炒粿条喜欢放鸭蛋，我和猫叔也试了一下，感觉味道还蛮好的！还有什么猪肠粉呀云吞面呀，想吃的统统来一份！走之前还要到对面喝一碗肉骨茶才肯走

吃饱喝足的两个“大肚子”等飞机回国了，槟城这个地方不大，但有太多惊喜了，下次我们会再回来的！
